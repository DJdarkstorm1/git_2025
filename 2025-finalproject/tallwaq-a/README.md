## 1一切的开始——ubuntu

### 遇到困难

- u盘无法格式化为fat32数据类型
- 找不到vmd开关
- bios面板第一次打开的方式不太一样
- 关闭vmd后不能打开win11
- 上不了无线网
- 驱动冲突，被迫重装ubuntu

### 解决问题

- 学习diskpart、磁盘管理器的用法，只分配不大于32g的空间
- 接触csdn社区，上面有很多对应的解决方案
- 取舍，有时候没有linux的网卡驱动，那就氪金买无线网卡吧
- 重装ubuntu的速度比之前快很多
