#第一节课：Shell、vim、脚本的基本使用
1. 学习、练习到shell中cd、ls、touch、mkdir的操作。
2. 学习到 使用vim编写脚本并且使用chmod赋予权限并运行。
3. 日志文件创建及写入的操作。
4. 查找文件并对其输出进行处理后输出。

#第二节课：Linux架构、命令行环境、数据整理方法
1. Linux系统的框架，主要以文件为内容形式。
2. bg、fg、jobs、暂停、停止的一些控制进程的操作。
3. 在编写脚本的过程中学会了更多的bash的语法，正则表达的语法（基本靠查）及正则表达的使用。

#第三节课：Git、gihub使用及规范
1. 学习了git的形式及github这个平台。
2. git的工作区、暂存区、本地仓库的作用，以及status仓库情况查询、增删查改、add暂存、commit提交、log日志b、版本回退、比较操作。
3. SSH Key的生成与在github远程仓库的使用。
4. 克隆仓库与推送拉取更新内容的远程仓库操作。
5. 分支的增删查及合并且切换操作，其作用及意义。
