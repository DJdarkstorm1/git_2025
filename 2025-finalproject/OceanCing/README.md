# 课程学习总结

## 1. Vim编辑器和Shell

### Vim

- 了解了Vim是命令行环境下一款强大的编辑器
- 学会了Vim的部分常用快捷键，模式切换等基本用法
- 学会了如何编辑`.vimrc`文件把Vim配置得更加个性化，如自动缩进，匹配符号等等

### Shell

- 学习了Shell的基本语法，如变量规则、符号使用规则、for循环、while循环等语句
- 学习了如何使用shell脚本编写命令脚本简化工作
- 学习了使用Vim编写shell脚本并运行

## 2. Linux系统架构和数据整理/筛选

### Linux系统架构

- 了解了Linux的文件系统和挂载规则
- 了解了Linux系统构造和交互环境

### 数据整理和筛选

- 通过作业需求学习了正则表达式，并学习了如何使用正则表达式筛选如日志内容等大体积信息
- 学习了几个常用正则表达式的shell命令如`sed`，`grep`等等，还学习到了如何使用`awk`命令提取字段，$赋值等等知识，并将他们运用到脚本当中，提高了获取信息的效率

## 3. Git的使用

### Git常用命令

- 学习了git本地仓库的一些常用命令比如`add`，`commit`，`gitlog`，学会了如何管理本地仓库
- 学习了本地仓库与远程仓库的关联和协同机制

### GitHub Flow的PR机制

- 了解了GitHub Flow工作流的PR机制，学习了如何开启Pull Request

- 学习了仓库分支`branch`的作用和分支管理如`merge`和`rebase`
- 了解了如何在团队协作中使用合理分支和PR

## 4. ROS框架编程

### catkin工作流

- 初步了解了ROS的catkin编译系统和工作流

### ROS的常用API

- 初步学习了ROS的常用函数、类的一些编程范式和编程方法、接口等
